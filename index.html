Irrational numbers

Numbers that cannot be written as p/q. Non-terminating, non-repeating decimals.

Irrational
π e √2 √3 √7
Rational
0.75 ⅓ √9 −2.5
Surd rules
√a × √b = √(ab)  |  √a ÷ √b = √(a/b)  |  1/√a = √a/a
Watch out √16 = 4 is rational. Only √n where n is not a perfect square is irrational.
2

Compound interest

Interest earned on principal plus previously earned interest.

GrowthA = P(1 + r/100)ⁿ
DepreciationA = P(1 − r/100)ⁿ
Simple interest (for comparison)I = (P × r × n) / 100
Example £2000 at 5% for 3 years → A = 2000 × 1.05³ = £2315.25. Interest = £315.25
  • Non-annual compounding: divide r by periods/year, multiply n
  • Interest earned = A − P
3

Probability

Basic probabilityP(A) = favourable outcomes / total outcomes
ComplementP(not A) = 1 − P(A)
Range0 ≤ P(A) ≤ 1
Independent (AND)P(A ∩ B) = P(A) × P(B)
Exclusive (OR)P(A ∪ B) = P(A) + P(B)
  • Tree diagrams: multiply along branches, add between branches
  • Without replacement: denominators decrease each pick
  • Expected frequency = n × P(A)
Example Bag: 3 red, 5 blue. P(2 red, no replacement) = 3/8 × 2/7 = 3/28
4

Functions

Rule mapping each input to exactly one output. Written f(x).

Composite — do inner first
fg(x) = f(g(x))  |  fg ≠ gf in general
Inverse — swap and solve
y = f(x) → swap x, y → solve for y → f⁻¹(x)
Example f(x) = 2x + 3. Swap: x = 2y + 3 → y = (x − 3)/2. So f⁻¹(x) = (x − 3)/2
  • Check: ff⁻¹(x) = x
  • Graph of f⁻¹ is f reflected in y = x
  • Domain = valid inputs | Range = possible outputs
5

Vectors

Quantity with magnitude and direction. Column form: (x over y).

Magnitude|a| = √(x² + y²)
Path ruleAB = b − a
Midpoint of AB½(a + b)
Parallel checkone is k × other
  • Addition / subtraction: component-wise
  • Scalar multiplication: multiply each component by k
  • Collinear: parallel vectors sharing a point
  • Parallelogram: opposite sides are equal vectors

Motion-time graphs

Distance–time
  • Gradient = speed
  • Flat line = stationary
  • Curve = changing speed
Speed–time
  • Gradient = acceleration
  • Area under = distance
  • Negative slope = decelerating
Area shapes for distance
Rectangle: b × h | Triangle: ½bh | Trapezium: ½(a + b)h
Speedspeed = distance / time
Accelerationa = Δv / t

Differentiation (extended only)

Power ruley = axⁿ → dy/dx = anxⁿ⁻¹
  • Constants differentiate to 0
  • Polynomials: differentiate term by term
  • Gradient at x = a: substitute a into dy/dx
  • Stationary points: solve dy/dx = 0
Second derivative test
d²y/dx² > 0 → minimum
d²y/dx² < 0 → maximum
Tangent at (x₁, y₁)y − y₁ = m(x − x₁), m = dy/dx
Normal at (x₁, y₁)same form, but gradient = −1/m
Example y = 2x³ − 4x² + 5x − 7 → dy/dx = 6x² − 8x + 5
Universal exam strategy
  • Show every step — method marks count
  • Keep exact values (fractions, surds, π) until the last step
  • Check units on motion questions
  • Re-read the question after solving — are you answering what was asked?

A vector has magnitude and direction. Column form stores it as \(\begin{pmatrix}x\\y\end{pmatrix}\).

Operations
Magnitude\(|\mathbf{a}| = \sqrt{x^{2} + y^{2}}\)
Path rule\(\overrightarrow{AB} = \mathbf{b} - \mathbf{a}\)
Midpoint of \(AB\)\(\tfrac{1}{2}(\mathbf{a} + \mathbf{b})\)
Parallel checkone is \(k \times\) other
Arithmetic
  • Addition / subtraction: component-wise
  • Scalar multiplication: multiply each component by \(k\)
  • Collinear: parallel vectors that share a point
  • Parallelogram: opposite sides are equal vectors
Example \(\mathbf{a} = (3, 4)\), \(\mathbf{b} = (-1, 2)\). \(\mathbf{a} + \mathbf{b} = (2, 6)\). \(\mathbf{a} - \mathbf{b} = (4, 2)\). \(2\mathbf{a} = (6, 8)\). \(|\mathbf{a}| = \sqrt{9+16} = 5\).
Position vectors (Extended) \(\overrightarrow{OA} = \mathbf{a}\), \(\overrightarrow{OB} = \mathbf{b}\). Point \(M\) divides \(AB\) in ratio \(2 : 3\). \(\overrightarrow{OM} = \mathbf{a} + \tfrac{2}{5}(\mathbf{b} - \mathbf{a}) = \tfrac{3}{5}\mathbf{a} + \tfrac{2}{5}\mathbf{b}\).

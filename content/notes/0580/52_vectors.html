A vector has magnitude and direction. Column form stores it as (x over y).

Operations
Magnitude|a| = √(x² + y²)
Path ruleAB = b − a
Midpoint of AB½(a + b)
Parallel checkone is k × other
Arithmetic
  • Addition / subtraction: component-wise
  • Scalar multiplication: multiply each component by k
  • Collinear: parallel vectors that share a point
  • Parallelogram: opposite sides are equal vectors
Example a = (3, 4), b = (−1, 2). a + b = (2, 6). a − b = (4, 2). 2a = (6, 8). |a| = √(9+16) = 5.
Position vectors (Extended) OA = a, OB = b. Point M divides AB in ratio 2 : 3. OM = a + (2/5)(b − a) = (3/5)a + (2/5)b.

Solve like equations — with one critical exception: multiplying or dividing by a negative flips the inequality.

Solving linear inequalities
Same moves as equations. Just flip ≤/≥/</> when × or ÷ by a negative.
Example 3 − 2x ≥ 11 → −2x ≥ 8 → x ≤ −4 (flipped).
Number line
Strict (<, >) → open circle ○ · Inclusive (≤, ≥) → filled circle ●
Double inequality
Example −1 < 2x − 3 ≤ 5 → add 3: 2 < 2x ≤ 8 → divide: 1 < x ≤ 4.
Graphical regions (Extended)
  • y < line → shade below; y > line → shade above
  • Strict → dashed line · Inclusive → solid line
  • Intersection of regions = set of points satisfying ALL inequalities

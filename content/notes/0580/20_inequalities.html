Solve like equations — with one critical exception: multiplying or dividing by a negative flips the inequality.

Solving linear inequalities
Same moves as equations. Just flip \(\le / \ge / < / >\) when \(\times\) or \(\div\) by a negative.
Example \(3 - 2x \ge 11\) → \(-2x \ge 8\) → \(x \le -4\) (flipped).
Number line
Strict (\(<\), \(>\)) → open circle ○ · Inclusive (\(\le\), \(\ge\)) → filled circle ●
Double inequality
Example \(-1 < 2x - 3 \le 5\) → add 3: \(2 < 2x \le 8\) → divide: \(1 < x \le 4\).
Graphical regions (Extended)
  • \(y < \text{line}\) → shade below; \(y > \text{line}\) → shade above
  • Strict → dashed line · Inclusive → solid line
  • Intersection of regions = set of points satisfying ALL inequalities

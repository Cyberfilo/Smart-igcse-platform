Probability measures how likely something is, between \(0\) (impossible) and \(1\) (certain).

Basic probability\(P(A) = \dfrac{\text{favourable outcomes}}{\text{total outcomes}}\)
Complement\(P(\text{not } A) = 1 - P(A)\)
Range\(0 \le P(A) \le 1\)
Sample space
List every equally likely outcome. Use a table for two-dice problems (\(36\) outcomes).
Example Single fair die. \(P(\text{prime}) = P(\{2,3,5\}) = \tfrac{3}{6} = \tfrac{1}{2}\). \(P(\text{not prime}) = 1 - \tfrac{1}{2} = \tfrac{1}{2}\).
  • Equally likely outcomes is an assumption — state it if unsure
  • Write probabilities as simplified fractions unless decimals asked
  • Tables and Venn diagrams often beat formula-only solutions
Watch units "Probability" is a number between \(0\) and \(1\), not a percentage or ratio. Convert before comparing.

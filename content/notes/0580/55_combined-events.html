Two events — multiply for "and", add for "or" (mutually exclusive). Tree diagrams make it visual.

Independent (AND)\(P(A \cap B) = P(A) \times P(B)\)
Exclusive (OR)\(P(A \cup B) = P(A) + P(B)\)
General OR\(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
Tree diagrams
  • Multiply along branches
  • Add between branches (different paths to the same outcome)
  • All outcomes at the end sum to \(1\) — useful check
Example (with replacement) Bag: \(3\) red, \(5\) blue. Pick \(2\) with replacement. \(P(\text{2 red}) = \tfrac{3}{8} \cdot \tfrac{3}{8} = \tfrac{9}{64}\).
Example (without replacement) Same bag without replacement. \(P(\text{2 red}) = \tfrac{3}{8} \cdot \tfrac{2}{7} = \tfrac{6}{56} = \tfrac{3}{28}\).
Exactly one red? "Red then blue" \(+\) "Blue then red" \(= P(R)P(B) + P(B)P(R)\). Both paths count.

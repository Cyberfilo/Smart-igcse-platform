Read the chart, extract the numbers, answer the exact question asked. Don't invent data.

What to check first
  • Axis labels AND units
  • Scale — is it linear? Does it start at 0 or further along?
  • Legend / key for multi-series charts
  • Title and caption for context
Comparing data sets
Always compare TWO numbers: a measure of centre (mean / median) AND a measure of spread (range / IQR).
Example Class A: mean 65, range 20. Class B: mean 65, range 50. Same average — but B is much more variable. Class A is more consistent.
Two-mark descriptions "On average, X did better" = 1 mark. "On average X did better (higher mean) but was less consistent (larger range)" = 2 marks. Centre + spread.

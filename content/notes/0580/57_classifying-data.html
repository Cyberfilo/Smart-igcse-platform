Two splits: qualitative vs quantitative, and discrete vs continuous. Pick chart type accordingly.

Data types
  • Qualitative: categories (colours, names, yes/no)
  • Quantitative discrete: countable (number of siblings, dice score)
  • Quantitative continuous: measurements (height, time, mass) — any value in a range
Chart that matches
Qualitativebar chart, pie chart, pictogram
Discretebar chart (gaps between bars)
Continuoushistogram (no gaps), frequency polygon
Example Shoe size data: quantitative discrete. Use bar chart, not histogram.
Grouped data Continuous data is often grouped into class intervals. Use class boundaries (e.g. \(149.5 \le h < 154.5\)), not limits (\(150 \le h < 155\)).

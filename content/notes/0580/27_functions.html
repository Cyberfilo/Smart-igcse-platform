(Extended.) A function maps each input to exactly one output. Written \(f(x)\).

Composite — do inner first
\(fg(x) = f(g(x))\)  |  \(fg \ne gf\) in general
Example \(f(x) = 2x + 1\), \(g(x) = x^{2}\). \(fg(3) = f(9) = 19\). \(gf(3) = g(7) = 49\). Not equal.
Inverse — swap and solve
\(y = f(x)\) → swap \(x, y\) → solve for \(y\) → that's \(f^{-1}(x)\)
Example \(f(x) = 2x + 3\). Swap: \(x = 2y + 3\) → \(y = \dfrac{x - 3}{2}\). So \(f^{-1}(x) = \dfrac{x - 3}{2}\).
  • Check: \(f\,f^{-1}(x) = x\) AND \(f^{-1}f(x) = x\)
  • Graph of \(f^{-1}\) is \(f\) reflected in \(y = x\)
  • Domain = valid inputs; Range = possible outputs
  • \(f(x)\) has an inverse only if it's one-to-one (no two inputs share an output)

A set is a collection of distinct elements. \(\xi\) is the universal set; \(A'\) is the complement of \(A\).

Notation
Member\(x \in A\) · \(x \notin A\)
Subset\(A \subseteq B\) · \(A \subset B\) (proper)
Union\(A \cup B\) = in \(A\) or \(B\)
Intersection\(A \cap B\) = in both
Empty\(\emptyset\) or \(\{\,\}\)
Count\(n(A)\) = size of \(A\)
Set builder
\(\{\, x : x \text{ is even},\ 0 < x \le 10 \,\} = \{2, 4, 6, 8, 10\}\)
Example \(\xi = \{1,\ldots,10\}\), \(A\) = multiples of \(2\), \(B\) = multiples of \(3\). \(A \cap B = \{6\}\). \(A \cup B = \{2,3,4,6,8,9,10\}\). \((A \cup B)' = \{1,5,7\}\).
Count rule \(n(A \cup B) = n(A) + n(B) - n(A \cap B)\). Always works; use it instead of counting twice.

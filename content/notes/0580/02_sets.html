A set is a collection of distinct elements. ξ is the universal set; A′ is the complement of A.

Notation
Memberx ∈ A · x ∉ A
SubsetA ⊆ B · A ⊂ B (proper)
UnionA ∪ B = in A or B
IntersectionA ∩ B = in both
Empty∅ or { }
Countn(A) = size of A
Set builder
{ x : x is even, 0 < x ≤ 10 } = {2, 4, 6, 8, 10}
Example ξ = {1…10}, A = multiples of 2, B = multiples of 3. A ∩ B = {6}. A ∪ B = {2,3,4,6,8,9,10}. (A ∪ B)′ = {1,5,7}.
Count rule n(A ∪ B) = n(A) + n(B) − n(A ∩ B). Always works; use it instead of counting twice.

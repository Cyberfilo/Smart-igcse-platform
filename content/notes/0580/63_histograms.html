(Extended.) Histograms with unequal class widths. Frequency density, not raw frequency, goes on the \(y\)-axis.

Frequency density\(\text{FD} = \dfrac{\text{frequency}}{\text{class width}}\)
Frequencyfrequency density \(\times\) class width (area of bar)
Why frequency density?
  • Wider class → more data space → artificially high raw frequency
  • Dividing by width corrects for this — bars now show true density
  • Area of each bar \(=\) frequency, so total area \(=\) total count
Example Class \(10 \le h < 20\) (width \(10\)) with frequency \(30\) → FD \(= 3\). Class \(20 \le h < 50\) (width \(30\)) with frequency \(60\) → FD \(= 2\).
Reading backwards Histogram bar has height (FD) \(= 4\), width \(5\) → frequency \(= 4 \times 5 = 20\).
No gaps Histogram bars touch (continuous data). If there's a gap, it's a bar chart, not a histogram.

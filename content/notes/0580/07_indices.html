Indices are just shorthand for repeated multiplication. Every rule comes from that idea.

Core rules
Multiply\(a^{m} \times a^{n} = a^{m+n}\)
Divide\(a^{m} \div a^{n} = a^{m-n}\)
Power of power\((a^{m})^{n} = a^{mn}\)
Zero\(a^{0} = 1\) (\(a \ne 0\))
Negative\(a^{-n} = \dfrac{1}{a^{n}}\)
Fractional (Ext)\(a^{m/n} = \sqrt[n]{a^{m}}\)
Extended \(8^{2/3} = (\sqrt[3]{8})^{2} = 2^{2} = 4\) · \(16^{-1/2} = \dfrac{1}{\sqrt{16}} = \tfrac{1}{4}\) · \(27^{-2/3} = \tfrac{1}{9}\)
Same base first To solve \(2^{x} = 32\), rewrite: \(2^{x} = 2^{5}\) → \(x = 5\). Rewriting to the same base unlocks most index equations.

Scale tells you how map distance relates to real distance. \(1 : n\) means 1 unit on paper = \(n\) real units.

Real distancemap distance \(\times\) scale factor
Map distancereal distance \(\div\) scale factor
Example Scale \(1 : 25\,000\). \(4.8\text{ cm}\) on map → \(4.8 \times 25\,000 = 120\,000\text{ cm} = 1.2\text{ km}\).
Bearings on a scale drawing
  • Bearings measured clockwise from North, always 3 digits (\(045°\), not \(45°\))
  • Use a protractor, not a ruler angle
  • For two-leg bearings questions: draw North at each turning point
Example Ship sails \(8\text{ km}\) on bearing \(060°\), then \(5\text{ km}\) on bearing \(150°\). Draw carefully with scale \(1\text{ cm} = 1\text{ km}\). Measure final distance + bearing from start.
Match units Convert everything to the same unit before scaling. \(25\,000\text{ cm} = 250\text{ m} = 0.25\text{ km}\) — easy to forget.

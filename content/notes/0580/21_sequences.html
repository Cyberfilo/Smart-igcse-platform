Three common families — linear (constant difference), quadratic (constant second difference), geometric (constant ratio).

Linear (arithmetic)
nth terma_n = dn + (a₁ − d) where d = common difference
Example 5, 8, 11, 14, …: d = 3, a₁ = 5 → a_n = 3n + 2.
Quadratic (Extended)
Second difference is constant. a_n = An² + Bn + C; 2A = second difference.
Example 4, 9, 18, 31, 48, … 1st diffs: 5, 9, 13, 17 · 2nd diff: 4 → A = 2 → a_n = 2n² + n + 1.
Geometric (Extended)
nth terma_n = a·r^(n−1) where r = common ratio
Example 2, 6, 18, 54, …: r = 3, a = 2 → a_n = 2 · 3^(n−1).
Diagnose first Check 1st difference — constant → linear. Not constant? Check 2nd difference — constant → quadratic. Check ratio — constant → geometric.

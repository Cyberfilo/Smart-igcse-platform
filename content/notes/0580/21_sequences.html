Three common families — linear (constant difference), quadratic (constant second difference), geometric (constant ratio).

Linear (arithmetic)
nth term\(a_{n} = dn + (a_{1} - d)\) where \(d\) is the common difference
Example \(5, 8, 11, 14, \ldots\): \(d = 3\), \(a_{1} = 5\) → \(a_{n} = 3n + 2\).
Quadratic (Extended)
Second difference is constant. \(a_{n} = An^{2} + Bn + C\); \(2A = \text{second difference}\).
Example \(4, 9, 18, 31, 48, \ldots\) 1st diffs: \(5, 9, 13, 17\) · 2nd diff: \(4\) → \(A = 2\) → \(a_{n} = 2n^{2} + n + 1\).
Geometric (Extended)
nth term\(a_{n} = a\,r^{n-1}\) where \(r\) is the common ratio
Example \(2, 6, 18, 54, \ldots\): \(r = 3\), \(a = 2\) → \(a_{n} = 2 \cdot 3^{n-1}\).
Diagnose first Check 1st difference — constant → linear. Not constant? Check 2nd difference — constant → quadratic. Check ratio — constant → geometric.

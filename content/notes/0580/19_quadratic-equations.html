(Extended.) Three methods — factorising (fastest when it works), quadratic formula (always works), completing the square (gives vertex form).

Quadratic formula
ax² + bx + c = 0 → x = (−b ± √(b² − 4ac)) / (2a)
DiscriminantΔ = b² − 4ac · >0 two roots · =0 one root · <0 no real roots
Factorising
Example x² − 5x + 6 = 0 → (x−2)(x−3) = 0 → x = 2 or 3.
Completing the square
x² + bx + c = (x + b/2)² − (b/2)² + c
Example x² + 6x − 7 = (x+3)² − 9 − 7 = (x+3)² − 16. Vertex at (−3, −16).
When to use what Integer coefficients and simple c → try factorising first. Ugly coefficients → formula. Question asks for vertex / max or min → complete the square.

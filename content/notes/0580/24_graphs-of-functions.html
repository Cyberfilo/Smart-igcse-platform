Five standard graph shapes you must recognise on sight.

Lineary = mx + c → straight line
Quadraticy = ax² + bx + c → parabola
Cubicy = ax³ + … → S-shape
Reciprocaly = k/x → hyperbola, asymptotes x=0, y=0
Exponentialy = aˣ → through (0, 1), never touches x-axis
Trig (Ext)y = sin x, cos x, tan x — periodic
Key features
  • x-intercepts: set y = 0
  • y-intercept: set x = 0
  • Parabola vertex: x = −b/(2a) — axis of symmetry
  • Exponential a > 1 → growth; 0 < a < 1 → decay
Example y = x² − 4x − 5 cuts x-axis where (x−5)(x+1) = 0 → x = 5 or x = −1. Axis: x = 2. Vertex: (2, −9).

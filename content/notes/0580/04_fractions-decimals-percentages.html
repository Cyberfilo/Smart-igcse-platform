Recurring decimals (Extended)
\(0.\dot{a} = \dfrac{a}{9}\) · \(0.\dot{a}\dot{b} = \dfrac{ab}{99}\) · \(0.\dot{a}\dot{b}\dot{c} = \dfrac{abc}{999}\)
Example Convert \(0.\dot{3}\dot{6}\) to a fraction. Let \(x = 0.363636\ldots\) → \(100x - x = 36\) → \(99x = 36\) → \(x = \tfrac{36}{99} = \tfrac{4}{11}\).
  • Terminating decimal ⇔ denominator (in lowest terms) has only \(2\)s and \(5\)s as prime factors
  • Always simplify the final fraction using HCF
  • \(\tfrac{1}{4} = 0.25 = 25\%\) · \(\tfrac{1}{3} = 0.\dot{3} = 33.\dot{3}\%\) · \(\tfrac{1}{8} = 0.125 = 12.5\%\) · \(\tfrac{1}{5} = 0.2 = 20\%\)

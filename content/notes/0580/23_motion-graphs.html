Distance–time
  • Gradient = speed
  • Flat line = stationary
  • Curve = changing speed (accelerating / decelerating)
  • Steeper = faster
Speed–time
  • Gradient = acceleration
  • Area under = distance travelled
  • Negative slope = decelerating
  • Flat line = constant speed
Area shapes for distance
Rectangle: \(b \times h\)  |  Triangle: \(\tfrac{1}{2}bh\)  |  Trapezium: \(\tfrac{1}{2}(a + b)h\)
Speed\(\text{speed} = \dfrac{\text{distance}}{\text{time}}\)
Acceleration\(a = \dfrac{\Delta v}{t}\)
Example Speed–time graph: accelerates \(0 \to 20\text{ m/s}\) in \(5\text{ s}\), holds \(20\text{ m/s}\) for \(10\text{ s}\), decelerates to \(0\) in \(5\text{ s}\). Total distance \(= \tfrac{1}{2}(5)(20) + 20(10) + \tfrac{1}{2}(5)(20) = 50 + 200 + 50 = 300\text{ m}\).
Don't confuse axes Gradient on d–t gives speed. Gradient on v–t gives acceleration. Area is only meaningful on v–t graphs.

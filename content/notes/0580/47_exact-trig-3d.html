(Extended.) Memorise three right triangles — they give exact values for \(30°\), \(45°\), \(60°\). 3D problems use Pythagoras + right-angle trig in a chosen plane.

Exact values
30°\(\sin = \tfrac{1}{2},\ \cos = \tfrac{\sqrt{3}}{2},\ \tan = \tfrac{1}{\sqrt{3}}\)
45°\(\sin = \cos = \tfrac{\sqrt{2}}{2},\ \tan = 1\)
60°\(\sin = \tfrac{\sqrt{3}}{2},\ \cos = \tfrac{1}{2},\ \tan = \sqrt{3}\)
0° / 90°\(\sin 0 = 0,\ \cos 0 = 1,\ \sin 90 = 1,\ \cos 90 = 0\)
3D problems
  1. Identify the RIGHT-angled triangle inside the 3D shape (often needs Pythagoras first)
  2. Redraw that 2D triangle, label sides
  3. Apply SOH CAH TOA or Pythagoras
Example Cuboid \(6 \times 8 \times 24\). Angle between space diagonal and base: base diagonal \(= \sqrt{36 + 64} = 10\). \(\tan\theta = \tfrac{24}{10}\) → \(\theta \approx 67.4°\).
Answer in exact form If a question asks for an exact value, don't put decimals — write \(\tfrac{\sqrt{3}}{2}\), not \(0.866\).

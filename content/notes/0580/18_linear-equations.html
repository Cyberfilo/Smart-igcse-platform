Inverse operations undo what's been done to x. Work from the outside in: +/− first, then ×/÷, then brackets.

Linear in one variable
Aim: ax + b = c → x = (c − b) / a
Example 5(x − 2) = 3x + 4 → 5x − 10 = 3x + 4 → 2x = 14 → x = 7.
Simultaneous equations
Eliminationscale so one variable matches, add/subtract
Substitutionsolve one for y, substitute into the other
Elimination 3x + 2y = 16, x − 2y = 0 → add: 4x = 16 → x = 4 → y = 2.
Fractions Multiply both sides by the LCM of denominators first — turns fractions into whole numbers.

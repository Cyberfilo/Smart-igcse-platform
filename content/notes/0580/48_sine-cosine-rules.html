(Extended.) Non-right triangles need sine or cosine rule. Pick based on what you're given.

Sine rule
a / sin A = b / sin B = c / sin C
Use whenYou know an angle and its opposite side, plus one more angle or side.
Cosine rule
a² = b² + c² − 2bc · cos A
Rearrangedcos A = (b² + c² − a²) / (2bc)
Use whenYou know all three sides, OR two sides and the included angle.
Area of a triangle
Area = ½ · a · b · sin C — two sides and the included angle
Example (cosine rule) a = 7, b = 5, C = 60°. c² = 49 + 25 − 70·cos 60° = 74 − 35 = 39. c = √39 ≈ 6.24.
Ambiguous case Sine rule can give two valid angles when solving for an angle (θ or 180 − θ). Sketch and check which makes sense.

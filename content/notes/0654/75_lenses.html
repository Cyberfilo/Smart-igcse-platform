A thin converging (convex) lens bends parallel rays to a focal point F. Ray diagrams predict image position.

Three principal rays
  1. Ray parallel to axis → refracts through F on the other side
  2. Ray through the centre → passes straight through
  3. Ray through near F → emerges parallel to axis
Image characteristics
Object beyond 2Freal, inverted, diminished
Object at 2Freal, inverted, same size
Object between F and 2Freal, inverted, magnified
Object at Fimage at infinity
Object closer than Fvirtual, upright, magnified (magnifying glass)
Magnification m = image height / object height. m = 2 → twice as tall.
Real vs virtual Real image: can be projected on a screen (rays actually converge). Virtual image: cannot (rays only appear to diverge from a point behind lens).

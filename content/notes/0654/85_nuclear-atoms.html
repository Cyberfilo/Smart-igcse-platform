Atomic nucleus has protons (Z) and neutrons (N). A = Z + N. Notation: ᴬ_Z X.

Key numbers
Z (atomic number)= number of protons = element identity
A (mass number)= protons + neutrons
NeutronsN = A − Z
Isotopes
Same Z, different N (and A). Same chemistry, different nuclear stability.
Nuclear size
  • Nucleus diameter ~10⁻¹⁵ m
  • Atom diameter ~10⁻¹⁰ m (100 000× larger)
  • Atom is mostly empty space — Rutherford's alpha scattering experiment
Nuclear stability
Nucleus stable when protons balanced by neutrons. Too many of either → unstable → radioactive decay.
Example Uranium-238: ²³⁸U has Z = 92, so N = 146. Unstable → alpha decay.
Rutherford's discovery Most α particles passed straight through gold foil, few bounced back. Proved: atom mostly empty, nucleus tiny, dense, positively charged.

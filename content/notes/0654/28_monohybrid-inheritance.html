One gene at a time. Dominant alleles (capital letter) mask recessive ones (lower case). Punnett squares predict offspring ratios.

Key terms
  • Genotype — the pair of alleles (e.g. Bb)
  • Phenotype — the visible characteristic (brown eyes)
  • Homozygous — same alleles (BB or bb)
  • Heterozygous — different alleles (Bb)
  • Dominant — shown even with one copy
  • Recessive — only shown when both copies present
Punnett square
Cross Bb × Bb → offspring genotypes: 1 BB, 2 Bb, 1 bb. Phenotype ratio: 3 brown : 1 blue.
Test cross An unknown brown-eyed parent crossed with bb (blue). If offspring are all brown → parent is BB. If half blue → parent is Bb.
Sex-linked inheritance (Extended)
Recessive allele on X chromosome (e.g. colour blindness, haemophilia). Males more often affected: only one X — no backup allele.

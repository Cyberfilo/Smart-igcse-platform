The nervous system detects stimuli and coordinates responses via electrical impulses along neurones.

Three neurone types
  • Sensory — from receptor to CNS
  • Relay — within the CNS, links others
  • Motor — from CNS to effector (muscle/gland)
Reflex arc
Stimulus → receptor → sensory neurone → relay (in spinal cord) → motor neurone → effector → response
Touching a hot object Heat → pain receptor in skin → sensory → spinal cord → motor → arm muscle contracts → hand pulls away. Bypasses brain for speed.
Synapses
  • Gap between two neurones
  • Neurotransmitter released, diffuses across, binds to receptor on next neurone
  • Ensures signal travels one way only
  • Target for many drugs
CNS vs PNS CNS = brain + spinal cord. PNS = everything else (sensory + motor neurones branching out).

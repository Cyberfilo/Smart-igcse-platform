Chemical formulas show ratios of atoms. Balanced equations obey conservation of atoms — same count each side.

Balancing rules
  1. Never change a formula (subscripts)
  2. Only add coefficients in front
  3. Balance atoms one element at a time
  4. Leave H and O until last if possible
Combustion of methane CH₄ + O₂ → CO₂ + H₂O. Balance: 1 CH₄ + 2 O₂ → 1 CO₂ + 2 H₂O. Check: C:1=1, H:4=4, O:4=4. ✓

Ionic equations (Extended)
Show only ions that change. Spectator ions cancel both sides.
Example Ag⁺(aq) + Cl⁻(aq) → AgCl(s). NO₃⁻ and Na⁺ spectators removed.
